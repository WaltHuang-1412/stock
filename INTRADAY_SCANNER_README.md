## 📊 核心邏輯

### 篩選條件1: 機會股
- 量比 > 3倍（爆量）
- 漲幅 < 3%（還沒大漲）
- 昨日法人買超 > 1,000K
- **推測**：法人吸貨中 ✅

### 篩選條件2: 投信聚焦
- 昨日投信買超 > 1,000K
- 今日爆量（量比 > 2倍）
- **判斷**：
  - 漲幅<3% → ✅ 吸貨中（機會）
  - 漲幅3-5% → ⚠️ 偏強（觀望）
  - 漲幅>5% → ❌ 已大漲（太晚）

### 篩選條件3: 法人對決
- 投信買超 > 1,000K
- 外資賣超 > 5,000K
- **判斷**：⚠️ 觀望（等盤後法人數據）

### 篩選條件4: 爆量下跌
- 量比 > 3倍
- 跌幅 < -2%
- **判斷**：❌ 避開（法人出貨）
