# AI 產業鏈深度分析筆記

**更新日期**：2026-03-12

---

## 一、AI 產業鏈總覽

```
┌─────────────────────────────────────────────────────────────┐
│                    【應用層】 Application                      │
│   ChatGPT / Copilot / 自動駕駛 / AI Agent / 企業AI           │
├─────────────────────────────────────────────────────────────┤
│                    【平台層】 Cloud & Model                    │
│   AWS / Azure / GCP / OpenAI / Anthropic / Meta AI           │
├─────────────────────────────────────────────────────────────┤
│                    【系統層】 System                           │
│   AI伺服器 / 網通設備 / 散熱 / 電源 / 機櫃                     │
├─────────────────────────────────────────────────────────────┤
│                    【零組件層】 Components                     │
│   PCB載板 / 連接器 / 光通訊模組 / Cable / 機構件               │
├─────────────────────────────────────────────────────────────┤
│                    【半導體層】 Semiconductor                  │
│   GPU/ASIC / HBM記憶體 / 先進封裝 / IC設計 / EDA              │
├─────────────────────────────────────────────────────────────┤
│                    【設備/材料層】 Equipment & Materials        │
│   光刻機 / 蝕刻 / 檢測 / 矽晶圓 / 光阻 / 特殊氣體             │
└─────────────────────────────────────────────────────────────┘
```

**核心邏輯**：資本支出（CapEx）從上往下流，營收從下往上堆。雲端巨頭砸錢 → 買伺服器 → 伺服器需要晶片+散熱+PCB → 晶片需要封裝+設備+材料。

## 二、半導體層（最上游）

### 2-1. AI 晶片（GPU / ASIC / AI加速器）

**這是整條產業鏈的核心驅動力。**

| 類型 | 全球龍頭 | 說明 |
|------|---------|------|

| **GPU #2** | AMD (AMD) | MI300X 搶食推論市場，性價比路線 |
| **訓練ASIC** | Google TPU | 自用為主，Cloud TPU 對外服務 |
| **推論ASIC** | Amazon Trainium/Inferentia | AWS 自用，降低對 NVIDIA 依賴 |
| **Custom ASIC** | Broadcom (AVGO) | 幫 Google/Meta/ByteDance 設計客製 AI 晶片 |
| **Custom ASIC** | Marvell (MRVL) | 幫 Amazon/Microsoft 設計客製晶片 |

## 三、零組件層

### 3-1. PCB 印刷電路板 / 載板（ABF Substrate）

**AI 伺服器的「神經系統」— 連接所有晶片的基板。**

| 類型 | 說明 | 技術門檻 |

| **ABF 載板** | 承載 CPU/GPU 的高階基板，層數多、線寬細 | 極高 |
| **HDI 板** | 高密度互連板，用於伺服器主板 | 高 |
| **CCL 銅箔基板** | PCB 的上游原材料 | 中 |
| **軟板 FPC** | 可撓式電路板 | 中高 |

**台股對應**：

| **台光電** | 2383 | CCL龍頭 | 高頻高速 CCL，AI伺服器 PCB 材料供應商（EPS 41.67 歷史新高） |
| **聯茂** | 6213 | CCL | 高頻 CCL 供應商 |

**投資邏輯**：

- **觀察指標**：台光電/聯茂營收（CCL需求）、欣興稼動率（載板需求）

### 3-2. 散熱模組

**AI 晶片功耗暴增（單顆 GPU 700W-1200W）→ 散熱成為關鍵瓶頸。**

| 技術路線 | 說明 | 發展階段 |
|---------|------|---------|
| **氣冷（傳統）** | 風扇+散熱片 | 成熟，但 AI 伺服器已不夠用 |
| **液冷（Direct Liquid Cooling）** | 冷卻液直接接觸晶片 | 主流趨勢，2025-2026 快速滲透 |
| **浸沒式液冷（Immersion）** | 整台伺服器泡在冷卻液中 | 下一代，技術仍在發展 |
| **水冷板（Cold Plate）** | 金屬板內走冷卻液 | 液冷核心零件，需求爆發中 |

**台股對應**：
| 台股 | 代號 | 角色 | 說明 |
|------|------|------|------|
| **奇鋐** | 3017 | 散熱龍頭 | 伺服器散熱+水冷板，EPS 49.17（歷史新高）水冷板產能擴 4-5 倍 |
| **建準** | 2421 | 散熱風扇 | 伺服器風扇龍頭 |
| **超眾** | 6230 | 散熱模組 | 熱導管+均熱片 |
| **雙鴻** | 3324 | 散熱模組 | 液冷散熱方案 |
| **健策** | 3653 | 水冷板 | AI 伺服器水冷板，產能擴張中 |
| **泰碩** | 3338 | 散熱 | 熱導管+散熱模組 |

**投資邏輯**：
- GPU 功耗每代翻倍 → 散熱 ASP 跟著翻倍
- 液冷滲透率從 <10% 快速拉升到 30-50%
- 水冷板是液冷最關鍵的零件（直接接觸晶片）
- **觀察指標**：液冷滲透率、NVIDIA 新 GPU 功耗規格、奇鋐/建準營收

## 八、當前關鍵趨勢（2026 Q1）

### 趨勢 1：GB200 NVL72 量產爬坡
- NVIDIA 下一代 AI 伺服器平台，單機櫃 72 顆 GPU
- 散熱需求暴增（全液冷設計）
- 台灣供應鏈全面受惠：鴻海（組裝）、奇鋐（液冷）、嘉澤（連接器）

### 趨勢 2：Custom ASIC 崛起
- 雲端巨頭不想被 NVIDIA 綁架 → 自研 AI 晶片
- Broadcom (+7.57%) + Marvell (+15.82%) = Custom ASIC 雙龍頭
- 台灣受惠：世芯-KY（ASIC 設計）、台積電（代工）

### 趨勢 3：CPO 共同封裝光學
- 傳統光模組插拔式 → CPO 直接封裝在晶片上
- 頻寬提升 + 功耗降低
- 台灣受惠：聯電（TFLN 矽光子）、聯亞（光晶片）

### 趨勢 4：AI 推論需求爆發
- 訓練 → 推論的轉移（ChatGPT/Copilot 用戶暴增）
- 推論需要大量中低階 GPU（不只 H100）
- 邊緣AI（手機/PC/車用）開始發力

### 趨勢 5：液冷滲透率快速拉升
- 2024: <10% → 2025: 20-30% → 2026: 40-50%
- 奇鋐水冷板產能擴 4-5 倍仍供不應求
- 健策切入水冷板市場

## 十、投資框架：如何看 AI 產業鏈

### 短期（1-3個月）看催化劑
- NVIDIA GTC（3月）→ 新產品發表 → AI 伺服器股
- 財報季（4月）→ 雲端巨頭 CapEx 指引 → 全產業鏈
- 記憶體報價（每月）→ DRAM/HBM 漲價 → 記憶體股

### 中期（3-12個月）看滲透率
- 液冷滲透率 → 散熱股
- 800G/1.6T 出貨量 → 光通訊股
- Custom ASIC 設計案 → ASIC 設計服務股

### 長期（1-3年）看技術轉折
- CPO 量產 → 光通訊革命
- AI 推論普及 → 邊緣 AI 晶片
- 機器人 + 具身智能 → 感測器/馬達/減速機

### 風險因素
1. **雲端巨頭縮減 CapEx** — 最大系統性風險（目前未發生）
2. **中美科技戰升級** — 出口管制擴大
3. **AI 泡沫論** — 營收不如預期時的估值修正
4. **供應鏈集中風險** — 台灣地緣政治
5. **技術迭代風險** — 新架構淘汰舊供應商
