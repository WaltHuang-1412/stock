# 五維度評分系統（Five Dimensions Scoring System）

**版本**：v1.0
**啟用日期**：2025-11-20
**目的**：解決「跌不買、漲不買」矛盾，從「一票否決」改為「綜合評分」

---

## 📋 系統概述

### 為什麼需要五維度評分？

**舊系統問題（一票否決制）**：
```
法人對決 → ❌ 不推薦（一票否決）
外資0張 → ❌ 不推薦（一票否決）
投信<10K → ❌ 不推薦（一票否決）

結果（2025/11/20）：
- 錯過12檔股票
- 12檔平均漲幅+2.8%
- 準確率損失：92.3%機會被忽略
```

**新系統優勢（綜合評分制）**：
```
五個維度同時評估：
1️⃣ 法人數據（30%）
2️⃣ 時事現況（30%）
3️⃣ 產業邏輯（20%）
4️⃣ 價格位置（10%）
5️⃣ 技術面（10%）

不是「一項不合格就不推薦」
而是「綜合得分決定推薦與否」
```

## 🎯 五大維度詳解

### 維度1️⃣：法人數據（30%權重）

**評分項目**：

| 指標 | 100分標準 | 80分標準 | 60分標準 | 40分標準 | 0分標準 |
|------|---------|---------|---------|---------|---------|
| 投信買超 | >20,000張 | 15,000-20,000 | 10,000-15,000 | 5,000-10,000 | <5,000或賣超 |
| 外資買超 | >15,000張 | 10,000-15,000 | 5,000-10,000 | 0-5,000 | 賣超或0張 |
| 三大法人 | >25,000張 | 15,000-25,000 | 5,000-15,000 | 0-5,000 | 賣超 |
| 法人一致性 | 三大同向買 | 投信+外資買 | 投外小分歧 | 投外對決 | 三大同向賣 |

**特殊情況處理**：

**外資主導型**（投信賣、外資大買）：
```
投信：-7,000張
外資：+17,000張
三大法人：+10,000張

判斷：外資主導（外資/投信比例 2.4:1）
評分：70分（外資力道強、可接受法人對決）
```

**投信主導型**（投信買、外資0張或小賣）：
```
投信：+8,000張
外資：0張
三大法人：+8,000張

判斷：投信主導、外資觀望
評分：60分（外資0張風險、但投信力道足）
```

**激烈對決型**（雙方都>20K）：
```
投信：+25,000張
外資：-39,000張
三大法人：-14,000張

判斷：激烈對決、無法判斷誰對誰錯
評分：40分（風險高）
```

---

### 維度2️⃣：時事與現況（30%權重）

**評分項目**：

**國際面**（50%）：
| 指標 | 100分 | 80分 | 60分 | 40分 | 0分 |
|------|------|------|------|------|-----|
| 美股表現 | 大漲>1% | 漲0.5-1% | 持平±0.5% | 跌-0.5~-1% | 大跌<-1% |
| 重大事件 | 超預期利多 | 符合預期利多 | 中性 | 符合預期利空 | 超預期利空 |
| 科技股 | NASDAQ+費半雙漲 | NASDAQ或費半漲 | 持平 | 下跌 | 大跌 |

**產業面**（50%）：
| 指標 | 100分 | 80分 | 60分 | 40分 | 0分 |
|------|------|------|------|------|-----|
| 產業新聞 | 重大利多明確 | 利多但不明確 | 中性 | 利空但不明確 | 重大利空 |
| 客戶動態 | 訂單爆增 | 訂單成長 | 持平 | 訂單減少 | 訂單崩潰 |
| 供應鏈 | 關鍵地位 | 重要環節 | 一般供應商 | 邊緣角色 | 被替代風險 |

**範例（2025/11/20）**：
```
國際面：
- NASDAQ反彈+0.6%（80分）
- 輝達財報超預期+4%（100分）
- 費半未提及（60分）
→ 國際面平均：80分

產業面：
- 輝達Blackwell爆單（100分）
- AI供應鏈需求明確（100分）
- 台積電ADR+1.08%（80分）
→ 產業面平均：93分

時事現況總分：(80×50% + 93×50%) = 86.5分
```

### 維度3️⃣：產業邏輯（20%權重）

**評分項目**：

| 評估面向 | 100分 | 80分 | 60分 | 40分 | 0分 |
|---------|------|------|------|------|-----|
| 產業趨勢 | 明確向上、多重催化 | 向上、單一催化 | 震盪、無明確方向 | 向下、短期反彈 | 產業衰退 |
| 龍頭地位 | 產業龍頭、護城河深 | 前三名、有優勢 | 中段班、無明顯優勢 | 後段班、競爭激烈 | 邊緣化、被淘汰 |
| 邏輯強度 | 基本面+政策+技術 | 基本面+政策 | 僅基本面 | 僅題材炒作 | 無邏輯 |
| 可驗證性 | 財報、訂單可驗證 | 產業數據可驗證 | 市場傳言 | 純猜測 | 無法驗證 |

**範例（南亞1303）**：
```
產業趨勢：
- 塑化產能利用率提升（80分）
- AI供應鏈需要塑化原料（80分）
- 油價穩定、成本可控（60分）
→ 趨勢分數：73分

龍頭地位：
- 台灣塑化龍頭（100分）
- 技術+產能護城河（100分）
→ 龍頭分數：100分

邏輯強度：
- 基本面（產能利用率）✅
- 產業面（AI需求）✅
- 可驗證（財報Q3/Q4）✅
→ 強度分數：100分

產業邏輯總分：(73 + 100 + 100) / 3 = 91分
```

---

### 維度4️⃣：價格位置（10%權重）

**評分項目**：

| 指標 | 100分 | 80分 | 60分 | 40分 | 0分 |
|------|------|------|------|------|-----|
| 近5日漲幅 | <3%（沒漲） | 3-5%（小漲） | 5-8%（中漲） | 8-12%（大漲） | >12%（暴漲） |
| 近20日位置 | 低檔整理 | 中段震盪 | 高檔整理 | 高檔突破 | 歷史高點 |
| 52週位置 | <30%（低位） | 30-50%（中低） | 50-70%（中高） | 70-90%（高位） | >90%（極高） |
| 技術型態 | 底部突破 | 整理後突破 | 趨勢中 | 高檔鈍化 | 乖離過大 |

**目的**：避免追高、找到合理價位

**範例（南亞1303，11/19收盤52元）**：
```
近5日漲幅：+3%（100分 - 沒大漲）
近20日：震盪整理（80分）
52週位置：約50%（80分 - 中段）
技術型態：橫盤突破（100分）

價格位置總分：(100 + 80 + 80 + 100) / 4 = 90分
```

---

### 維度5️⃣：技術面量能（10%權重）

**評分項目**：

| 指標 | 100分 | 80分 | 60分 | 40分 | 0分 |
|------|------|------|------|------|-----|
| 量比 | 1.5-2.5x（健康放量） | 1.0-1.5x（溫和） | 0.7-1.0x（正常） | 0.5-0.7x（縮量） | >3x（爆量風險）或<0.5x |
| 量價關係 | 量增價漲 | 量平價漲 | 量增價平 | 量縮價漲 | 量增價跌 |
| 趨勢 | 突破整理區 | 沿上升軌道 | 橫盤震盪 | 沿下降軌道 | 破底 |
| 支撐壓力 | 突破壓力、遠離支撐 | 靠近支撐 | 壓力支撐之間 | 靠近壓力 | 跌破支撐 |

**目的**：確認進場時機、避免量能異常

**範例（南亞1303盤中）**：
```
量比：0.7x（60分 - 正常，未爆量）
量價關係：量平價漲（80分）
趨勢：橫盤突破（80分）
支撐壓力：突破整理區（100分）

技術面總分：(60 + 80 + 80 + 100) / 4 = 80分
```

## 🎲 綜合評分計算

### 標準權重（強勢市場）

```
總分 = 法人數據×30% + 時事現況×30% + 產業邏輯×20% + 價格位置×10% + 技術面×10%
```

**範例（南亞1303）**：
```
法人數據：70分（外資主導、投信賣超）
時事現況：86.5分（輝達利多、NASDAQ反彈）
產業邏輯：91分（塑化復甦、AI供應鏈）
價格位置：90分（52元合理、未追高）
技術面：80分（量比正常、突破整理）

總分 = 70×30% + 86.5×30% + 91×20% + 90×10% + 80×10%
     = 21 + 25.95 + 18.2 + 9 + 8
     = 82.15分
```

### 權重調整（不同市場環境）

**強勢市場**（輝達利多、NASDAQ大漲）：
```
時事現況：30%（提高）
法人數據：30%
產業邏輯：20%
價格位置：10%
技術面：10%

理由：時事利多主導、法人對決影響降低
```

**弱勢市場**（大跌、恐慌）：
```
法人數據：40%（提高）
時事現況：30%
產業邏輯：10%（降低）
價格位置：10%
技術面：10%

理由：法人逃命最重要、產業邏輯無用
```

**震盪市場**（無明確方向）：
```
法人數據：35%
產業邏輯：25%（提高）
時事現況：20%（降低）
價格位置：10%
技術面：10%

理由：需要基本面支撐、時事影響小
```

## 📊 推薦門檻

### 評分對應推薦策略

| 總分區間 | 推薦等級 | 倉位建議 | 星等 | 策略 |
|---------|---------|---------|------|------|
| **≥85分** | 強烈推薦 | 15-20% | ⭐⭐⭐⭐⭐ | 重倉進場 |
| **75-84分** | 推薦 | 10-15% | ⭐⭐⭐⭐ | 標準倉位 |
| **65-74分** | 可考慮 | 5-10% | ⭐⭐⭐ | 小倉位試探 |
| **55-64分** | 觀望優先 | 3-5% | ⭐⭐ | 極小倉位 |
| **45-54分** | 不推薦 | 0% | ⭐ | 觀望 |
| **<45分** | 避開 | 0% | ❌ | 絕對不碰 |

### 特殊情況調整

**單一維度極低分（<30分）觸發警示**：
```
即使總分>70分，若任一維度<30分 → 降級處理

範例：
總分：72分
但法人數據：25分（三大法人大賣超）
→ 降級為「觀望優先」（55-64分等級）

理由：法人大逃不能忽視
```

**單一維度極高分（>95分）加分獎勵**：
```
若任一維度>95分 → 總分+3分獎勵

範例：
總分：77分
時事現況：98分（重大利多）
→ 調整為80分（+3分獎勵）
→ 等級提升至「強烈推薦」

理由：重大利多可壓過其他疑慮
```

**Step 2：逐一評分**
```
1️⃣ 法人數據：投信+外資+自營商+一致性 → /100分
2️⃣ 時事現況：國際面+產業面 → /100分
3️⃣ 產業邏輯：趨勢+龍頭+邏輯強度 → /100分
4️⃣ 價格位置：近5日+近20日+52週+技術 → /100分
5️⃣ 技術面：量比+量價+趨勢+支撐壓力 → /100分
```

**Step 3：計算總分**
```
判斷市場環境 → 選擇權重
總分 = Σ(各維度分數 × 權重)
```

**Step 4：對照推薦門檻**
```
≥85分 → 強烈推薦15-20%
75-84分 → 推薦10-15%
65-74分 → 可考慮5-10%
55-64分 → 觀望優先3-5%
<55分 → 不推薦或避開
```

## ⚠️ 注意事項

### 系統限制

1. **評分有主觀性**：不同分析師可能給出不同分數
2. **權重需調整**：不同市場環境需手動調整權重
3. **不是絕對**：高分也可能虧損、低分也可能漲

### 使用原則

1. **透明化評分**：每個維度都要說明理由
2. **一致性**：同類股票用相同標準
3. **可驗證**：評分依據要可查證（法人數據、新聞連結）
4. **持續校準**：每週檢討評分準確度、調整標準

### 禁止行為

❌ **事後調整評分**：不能「今天漲了」就回頭改評分
❌ **選擇性評分**：不能「只評看好的」忽略看壞的
❌ **忽略低分維度**：不能「總分高」就忽略某維度極低分
❌ **盲目相信分數**：分數是參考、最終仍需人工判斷
