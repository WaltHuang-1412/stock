# 五維度評分系統（Five Dimensions Scoring System）

**版本**：v3.0（新增時事先行流程）
**啟用日期**：2025-12-17
**目的**：解決「推薦股都在漲」問題，從「法人先行」改為「時事先行」選股邏輯

---

## 📋 系統概述

### 為什麼需要五維度評分？

**舊系統問題（一票否決制）**：
```
法人對決 → ❌ 不推薦（一票否決）
外資0張 → ❌ 不推薦（一票否決）
投信<10K → ❌ 不推薦（一票否決）

結果（2025/11/20）：
- 錯過12檔股票
- 12檔平均漲幅+2.8%
- 準確率損失：92.3%機會被忽略
```

**v2.0系統問題（法人先行）**：
```
⚠️ 重大問題（2025/12/17發現）：
- 規則說：法人30% + 時事30%
- 實際執行：法人70-80%、時事10-20%
- 結果：金融股佔推薦75-80%，推薦股都「已經在漲」

原因：從「法人買超TOP10」選股，時事只是事後解釋
```

**新系統優勢（時事先行 v3.0）**：
```
五個維度同時評估：
1️⃣ 法人數據（30%）← 用於驗證，不是選股
2️⃣ 時事現況（30%）← 第一步：決定產業方向
3️⃣ 產業邏輯（20%）
4️⃣ 價格位置（10%）
5️⃣ 技術面（10%）

🔥 關鍵改變：
- 舊流程：法人TOP10 → 選股 → 時事解釋
- 新流程：時事 → 產業 → 個股 → 法人驗證
```

## 🔥 時事先行選股流程（v3.0核心）

### 為什麼要「時事先行」？

**問題診斷（2025/12/17）**：
```
用戶反饋：「操 你他媽不要一直給我正在漲的」
「討論 為什麼現在的盤前與盤中一直只給我看漲的，然後盤後繼續事後諸葛」

根本原因調查：
- 12/12-12/17 金融股佔推薦 75-80%
- 每天推薦的股票「開盤就已經在漲」
- 用戶無法進場

錯誤流程：
法人買超TOP10 → 選股（金融股居多）→ 用時事解釋 → 推薦
          ↑
    這裡就決定了股票範圍，時事只是事後補充
```

**解決方案**：
```
正確流程（時事先行）：
時事分析 → 判斷產業催化 → 展開產業個股 → 法人驗證 → 價格篩選 → 最終推薦
    ↑
  這裡決定產業方向，法人只是驗證
```

---

### 時事先行四步驟

**Step 1: 時事分析 → 判斷產業催化**

```markdown
🔥 時事→產業對照表（必須使用）：

| 時事訊號 | 利多產業 | 代表股票 |
|---------|---------|---------|
| 費半漲>1% | 半導體 | 台積電、聯電、日月光、欣興、聯發科 |
| 費半跌>1% | 金融（防禦）| 凱基金、兆豐金、台新金、中信金 |
| 輝達漲>3% | AI伺服器+載板 | 廣達、緯創、欣興、景碩 |
| DRAM漲價 | 記憶體 | 南亞科、華邦電、旺宏 |
| 原油漲>3% | 塑化+航運 | 台塑、南亞、長榮、陽明 |
| 美元升值 | 電子出口 | 台積電、聯發科、鴻海 |
| 美元貶值 | 金融+內需 | 金融股、統一、台泥 |
| 蘋果利多 | 蘋概股 | 鴻海、大立光、台郡、玉晶光 |
| 電動車利多 | 汽車零組件 | 裕隆、和泰、貿聯-KY |
```

**Step 2: 產業展開 → 列出候選股票**

```markdown
範例（假設今日費半+1.5%）：

時事判斷：費半+1.5% → 利多半導體
產業展開：
├─ 晶圓代工：台積電(2330)、聯電(2303)
├─ 封測：日月光(3711)
├─ 載板：欣興(3037)、景碩(3189)
├─ IC設計：聯發科(2454)、瑞昱(2379)
└─ 設備：弘塑(3131)

候選股票：7檔（全部來自時事判斷的產業）
```

**Step 3: 法人驗證 → 篩選有法人支持的股票**

```bash
# 對候選股票查詢法人數據
python3 scripts/check_institutional.py 2330 20251216
python3 scripts/check_institutional.py 2303 20251216
...

# 篩選標準：
# ✅ 法人買超 > 5,000張 → 保留
# ⚠️ 法人 -5,000 ~ +5,000 → 觀望
# ❌ 法人賣超 > 5,000張 → 移除
```

**Step 4: 價格篩選 → 排除已大漲股票**

```markdown
價格篩選規則：
✅ 近5日漲幅 < 3% → 優先推薦
⚠️ 近5日漲幅 3-8% → 可推薦（標註風險）
❌ 近5日漲幅 > 8% → 不推薦（追高風險）
```

### 時事先行 vs 法人先行 對比

| 比較項目 | 法人先行（舊） | 時事先行（新） |
|---------|--------------|--------------|
| **第一步** | 查法人TOP10 | 分析時事/國際市場 |
| **選股來源** | 法人買超名單 | 時事利多產業 |
| **法人角色** | 選股依據 | 驗證工具 |
| **時事角色** | 事後解釋 | 選股依據 |
| **產業分布** | 可能集中（如75%金融）| 強制分散（≤50%） |
| **價格狀態** | 可能已漲（法人已買） | 較可能還沒漲 |

---

### 產業分散規則（v3.0強制）

```markdown
🔴 產業分散規則（必須遵守）：

| 規則 | 說明 |
|------|------|
| 單一產業上限 | ≤50%（如5檔推薦，金融最多2-3檔）|
| 最少產業數 | ≥2個不同產業 |
| 同質性警示 | 同產業≥3檔需標註「產業集中風險」|

範例：
❌ 錯誤：推薦5檔，4檔金融股（80%）
✅ 正確：推薦5檔，2檔金融+2檔半導體+1檔傳產
```

## 🎯 五大維度詳解

### 維度1️⃣：法人數據（30%權重）

**評分項目**：

| 指標 | 100分標準 | 80分標準 | 60分標準 | 40分標準 | 0分標準 |
|------|---------|---------|---------|---------|---------|
| 投信買超 | >20,000張 | 15,000-20,000 | 10,000-15,000 | 5,000-10,000 | <5,000或賣超 |
| 外資買超 | >15,000張 | 10,000-15,000 | 5,000-10,000 | 0-5,000 | 賣超或0張 |
| 三大法人 | >25,000張 | 15,000-25,000 | 5,000-15,000 | 0-5,000 | 賣超 |
| 法人一致性 | 三大同向買 | 投信+外資買 | 投外小分歧 | 投外對決 | 三大同向賣 |

**特殊情況處理**：

**外資主導型**（投信賣、外資大買）：
```
投信：-7,000張
外資：+17,000張
三大法人：+10,000張

判斷：外資主導（外資/投信比例 2.4:1）
評分：70分（外資力道強、可接受法人對決）
```

**投信主導型**（投信買、外資0張或小賣）：
```
投信：+8,000張
外資：0張
三大法人：+8,000張

判斷：投信主導、外資觀望
評分：60分（外資0張風險、但投信力道足）
```

**激烈對決型**（雙方都>20K）：
```
投信：+25,000張
外資：-39,000張
三大法人：-14,000張

判斷：激烈對決、無法判斷誰對誰錯
評分：40分（風險高）
```

### 維度2️⃣：時事與現況（30%權重）

**評分項目**：

**國際面**（50%）：
| 指標 | 100分 | 80分 | 60分 | 40分 | 0分 |
|------|------|------|------|------|-----|

| 重大事件 | 超預期利多 | 符合預期利多 | 中性 | 符合預期利空 | 超預期利空 |
| 科技股 | NASDAQ+費半雙漲 | NASDAQ或費半漲 | 持平 | 下跌 | 大跌 |

**產業面**（50%）：
| 指標 | 100分 | 80分 | 60分 | 40分 | 0分 |
|------|------|------|------|------|-----|
| 產業新聞 | 重大利多明確 | 利多但不明確 | 中性 | 利空但不明確 | 重大利空 |
| 客戶動態 | 訂單爆增 | 訂單成長 | 持平 | 訂單減少 | 訂單崩潰 |
| 供應鏈 | 關鍵地位 | 重要環節 | 一般供應商 | 邊緣角色 | 被替代風險 |

**範例（2025/11/20）**：

時事現況總分：(80×50% + 93×50%) = 86.5分
```

## 🎲 綜合評分計算

### 標準權重（強勢市場）

```
總分 = 法人數據×30% + 時事現況×30% + 產業邏輯×20% + 價格位置×10% + 技術面×10%
```

**範例（南亞1303）**：
```
法人數據：70分（外資主導、投信賣超）
時事現況：86.5分（輝達利多、NASDAQ反彈）
產業邏輯：91分（塑化復甦、AI供應鏈）
價格位置：90分（52元合理、未追高）
技術面：80分（量比正常、突破整理）

總分 = 70×30% + 86.5×30% + 91×20% + 90×10% + 80×10%
     = 21 + 25.95 + 18.2 + 9 + 8
     = 82.15分
```

### 權重調整（不同市場環境）

**強勢市場**（輝達利多、NASDAQ大漲）：
```
時事現況：30%（提高）
法人數據：30%
產業邏輯：20%
價格位置：10%
技術面：10%

理由：時事利多主導、法人對決影響降低
```

**弱勢市場**（大跌、恐慌）：
```
法人數據：40%（提高）
時事現況：30%
產業邏輯：10%（降低）
價格位置：10%
技術面：10%

理由：法人逃命最重要、產業邏輯無用
```

**震盪市場**（無明確方向）：
```
法人數據：35%
產業邏輯：25%（提高）
時事現況：20%（降低）
價格位置：10%
技術面：10%

理由：需要基本面支撐、時事影響小
```

## 📊 推薦門檻

### 評分對應推薦策略

| 總分區間 | 推薦等級 | 倉位建議 | 星等 | 策略 |
|---------|---------|---------|------|------|
| **≥85分** | 強烈推薦 | 15-20% | ⭐⭐⭐⭐⭐ | 重倉進場 |
| **75-84分** | 推薦 | 10-15% | ⭐⭐⭐⭐ | 標準倉位 |
| **65-74分** | 可考慮 | 5-10% | ⭐⭐⭐ | 小倉位試探 |
| **55-64分** | 觀望優先 | 3-5% | ⭐⭐ | 極小倉位 |
| **45-54分** | 不推薦 | 0% | ⭐ | 觀望 |
| **<45分** | 避開 | 0% | ❌ | 絕對不碰 |

### 特殊情況調整

**單一維度極低分（<30分）觸發警示**：
```
即使總分>70分，若任一維度<30分 → 降級處理

範例：
總分：72分
但法人數據：25分（三大法人大賣超）
→ 降級為「觀望優先」（55-64分等級）

理由：法人大逃不能忽視
```

**單一維度極高分（>95分）加分獎勵**：
```
若任一維度>95分 → 總分+3分獎勵

範例：
總分：77分
時事現況：98分（重大利多）
→ 調整為80分（+3分獎勵）
→ 等級提升至「強烈推薦」

理由：重大利多可壓過其他疑慮
```

## ⚠️ 注意事項

### 系統限制

1. **評分有主觀性**：不同分析師可能給出不同分數
2. **權重需調整**：不同市場環境需手動調整權重
3. **不是絕對**：高分也可能虧損、低分也可能漲

### 使用原則

1. **透明化評分**：每個維度都要說明理由
2. **一致性**：同類股票用相同標準
3. **可驗證**：評分依據要可查證（法人數據、新聞連結）
4. **持續校準**：每週檢討評分準確度、調整標準

### 禁止行為

❌ **事後調整評分**：不能「今天漲了」就回頭改評分
❌ **選擇性評分**：不能「只評看好的」忽略看壞的
❌ **忽略低分維度**：不能「總分高」就忽略某維度極低分
❌ **盲目相信分數**：分數是參考、最終仍需人工判斷

## 🔥 雙層推薦系統（v2.0新增 2025-11-25）

### 為什麼需要雙層系統？

**用戶反饋（2025/11/25）**：
```
用戶：「欣興真的獲利也不是你推薦的，我還想問為什麼推薦的都沒什麼用」

實際數據：
- 我的推薦：永豐金+0.94%、彰銀+1.00%（穩健但漲幅小）
- 用戶自選：欣興+9.50%（短線爆發）

問題診斷：
我只推薦「穩健型」（法人主導、1-3月+10%）
用戶也想要「爆發型」（產業催化、1-3天+10%）
```

**解決方案**：雙層推薦系統 = 穩健型 + 爆發型

---

### 雙層系統核心差異

| 類型 | 範例 | 漲幅 | 時間週期 | 主要驅動 | 權重分配 |

| **穩健型** | 永豐金、彰銀 | +1-3% | 1-3個月 | 法人持續買超 | 法人30%、時事30% |
| **爆發型** | 欣興 | +10%+ | 1-3天 | 產業催化+突然爆量 | 時事35%、產業30% |

產業邏輯：20%

技術面：10%
```

**推薦門檻**：

75-84分 → 推薦10-15% ⭐⭐⭐⭐
65-74分 → 可考慮5-10% ⭐⭐⭐
<65分 → 不推薦
```

**目標**：1-3個月+10-20%收益、法人主導、風險低

### 爆發型評分系統（v2.0新增）

**權重配置**：
```
時事現況：35%  ← 最重要（產業催化劑、突發利多）
產業邏輯：30%  ← 第二重要（爆發邏輯強度）
技術面：15%    ← 量能異動（盤中爆量）
法人數據：15%  ← 降低權重（散戶也能推動）
價格位置：5%   ← 爆發股不在乎已漲5%
```

**推薦門檻**（比穩健型更嚴格）：
```
≥80分 → 強烈推薦10-15% ⭐⭐⭐⭐⭐ （風險高、倉位降低）
70-79分 → 推薦5-10% ⭐⭐⭐⭐
60-69分 → 可考慮3-5% ⭐⭐⭐ （試單）
<60分 → 不推薦
```

**目標**：1-3天+10%+收益、產業驅動、風險高

### 爆發型五維度詳解

#### 維度1️⃣：時事現況（35%權重）← **爆發型最關鍵**

**評分標準**：

| 分數區間 | 條件 | 範例 |
|---------|------|------|
| **30-35分** | 重大產業催化劑（當日/隔日生效） | 輝達財報超預期+50%、蘋果發表會、突發訂單 |
| **24-29分** | 明確產業利多（1週內生效） | 記憶體漲價、被動元件缺貨、客戶財報利多 |
| **18-23分** | 產業趨勢延續 | AI需求持續、電動車滲透率上升 |
| **12-17分** | 產業中性 | 無明確利多或利空 |
| **<12分** | 產業利空 | 客戶砍單、產能過剩、價格崩跌 |

**範例（欣興 2025/11/25）**：
```
時事現況評分：31/35分

國際面（50%）：
- 輝達Blackwell爆單（100分）
- AI伺服器需求強勁（90分）
→ 國際面：95分

產業面（50%）：
- ABF載板供不應求（100分）
- 台積電CoWoS需求帶動（90分）
- 大客戶訂單明確（100分）
→ 產業面：97分

時事現況總分：(95×50% + 97×50%) × 35% = 32分 → 調整為31分（扣除盤中已漲）
```

---

#### 維度2️⃣：產業邏輯（30%權重）← **爆發型第二關鍵**

**評分標準**：

| 分數區間 | 條件 | 範例 |
|---------|------|------|
| **25-30分** | 龍頭+獨家供應+爆發邏輯明確 | 台積電CoWoS、欣興ABF載板、創意ASIC設計 |
| **19-24分** | 產業前3名+明確邏輯 | 聯電先進封裝、南亞AI供應鏈 |
| **13-18分** | 產業相關但非龍頭 | 力積電記憶體、華邦電DRAM |
| **7-12分** | 邊緣供應商 | 非關鍵零組件、可替代性高 |
| **<7分** | 邏輯薄弱或錯誤 | 純題材炒作、無實質訂單 |

**爆發邏輯三要素**：
1. **不可替代性**：只有這家能做 or 產能緊缺
2. **爆發時機**：催化劑即將發生（財報、訂單、產能開出）
3. **可驗證性**：能用財報、新聞、客戶動態驗證

**範例（欣興 2025/11/25）**：
```
產業邏輯評分：28/30分

龍頭地位：
- ABF載板台灣前2名（100分）
- 技術護城河深（高階ABF）（100分）

可驗證性：
- 台積電法說會提及（✅）
- 客戶輝達財報證實（✅）
- Q4財報可驗證（✅）

產業邏輯總分：(100 + 100 + 97) / 3 × 30% = 30分 → 調整為28分（保守）
```

#### 維度3️⃣：技術面（15%權重）← **爆發型獨特指標**

**評分標準**：

| 分數區間 | 條件 | 範例 |
|---------|------|------|
| **12-15分** | 盤中爆量3x+強勢突破+買盤>60% | 欣興12:30爆量+5%、買盤61.8% |

| **3-5分** | 量縮 | 量比<0.8x |
| **<3分** | 爆量下跌 | 量比3x+跌停（出貨） |

**盤中籌碼分析**：
```
買盤>60% → 15分（強勢吸貨）
買盤50-60% → 12分（穩健）
買盤40-50% → 9分（中性）
買盤<40% → 3分（賣壓重）

#### 維度4️⃣：法人數據（15%權重）← **爆發型降低權重**

**為什麼降低權重？**
```
爆發型股票特點：
- 散戶追價也能推動（短線爆發不一定需要法人）
- 產業催化劑>法人態度（時事決定短線走勢）
- 法人數據落後（昨日數據、盤中無法取得）

因此：法人數據從穩健型30% → 爆發型15%
```

**評分標準**（與穩健型相同、但權重降低）：

| 分數區間 | 條件 | 範例 |
|---------|------|------|
| **13-15分** | 投信+外資一致大買超>20K | 三大法人+25K |
| **10-12分** | 投信或外資單邊大買超>15K | 投信+15K、外資0 |
| **7-9分** | 法人小買超5-15K | 三大法人+8K |
| **4-6分** | 法人中性±5K | 略買或略賣 |
| **<4分** | 法人大賣超 | 三大法人-10K |

**範例（欣興 2025/11/25，使用11/24數據）**：
```
法人數據評分：9/15分

11/24法人數據：
- 投信：+266張（微量）（50分）
- 外資：0張（⚠️）（20分）
- 自營：+268張（60分）
- 三大：+2,057張（小買超）（60分）

評分理由：
- 投信力道極弱（僅+266張）
- 外資0張風險未解除
- 但三大小買超+2,057張
→ 法人數據評分：(50 + 20 + 60 + 60) / 4 = 47.5分 → 47.5 × 15% = 7分

⚠️ 若是穩健型，這種數據會拿40-50分（不推薦）
✅ 但爆發型重視時事+產業，法人數據降低權重至15%，7分可接受
```

---

#### 維度5️⃣：價格位置（5%權重）← **爆發型最不重要**

**為什麼降低權重？**
```
爆發型股票特點：
- 已漲5%不是問題（產業催化劑強勁、可能繼續漲）
- 追高風險由停損控制（快速停損-5%）
- 重點是「爆發邏輯」而非「便宜價位」

因此：價格位置從穩健型10% → 爆發型5%
```

**評分標準**（調整後）：

| 分數區間 | 條件 | 範例 |
|---------|------|------|
| **4-5分** | 近5日<3%（沒漲） | 整理後突破 |
| **3分** | 近5日3-8%（小漲可接受） | 欣興+5%（產業催化強） |
| **2分** | 近5日8-15%（中漲警戒） | 已漲一段、風險增加 |
| **<2分** | 近5日>15%（大漲不追） | 追高陷阱 |

**範例（欣興 2025/11/25）**：
```
價格位置評分：3/5分

近期漲幅：
- 近5日：約+3%（小漲）（60分）
- 盤中：+5.06%（中漲）（40分）

評分理由：
- 盤中已漲+5.06%（若用穩健型評分僅30分）
- 但產業催化強勁（輝達利多+ABF缺貨）
- 爆發型可接受小追高
→ 價格位置評分：60分 → 60 × 5% = 3分
```

**公式**：
```
總分 = 時事現況×35% + 產業邏輯×30% + 技術面×15% + 法人數據×15% + 價格位置×5%
```

```

```
