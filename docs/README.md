### 優化 B：產業展開自動深度（expand_industry.py --auto）

**功能**：
- 根據美股數據自動判斷展開深度（0-3）
- 催化 ≥5% → depth 3（強烈，完整產業鏈）
- 催化 2-5% → depth 2（中等，核心+供應鏈）
- 催化 0.5-2% → depth 1（微弱，核心+第一線）
- 催化 <0.5% → depth 0（幾乎無，僅核心）

**效果**：
- +3% 準確率
- 避免人工誤判深度
