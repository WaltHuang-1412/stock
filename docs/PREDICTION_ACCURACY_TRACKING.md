```

## 🎯 實戰建議

### ✅ 什麼時候跟著法人買超走

1. **產業趨勢明確時**（準確率70%+）
   - 例：DRAM漲價、AI訂單、面板反彈
   - 法人買超 + 產業邏輯 = 高勝率

2. **連續買超時**
   - 力積電連3日買超 → 10/28成功
   - 友達連2日買超 → 10/29成功

3. **買超>10K張時**
   - 力道夠強，不是試探性買超

### ❌ 什麼時候不跟法人買超

1. **無產業邏輯時**（準確率40-50%）
   - 金融股買超但產業弱勢
   - 個股買超但無催化劑

2. **單日試探性買超**
   - 買超<5K張
   - 昨買今賣的誘多陷阱（康舒案例）

3. **市場環境不佳時**
   - 川習會等重大事件前
   - 大盤系統性風險

```

```

**2. 產業分類精準**
```

- 純銀行（彰銀）：受惠利差

```

```
不再說「等回檔」「開盤進場」（模糊推薦）
