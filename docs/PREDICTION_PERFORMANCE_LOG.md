## 🎯 追蹤原則

1. **客觀記錄**：
   - 所有預測必須有時間戳記
   - 進場價、目標價、停損必須明確
   - 不修改歷史預測

2. **停損紀律**：
   - 觸發停損（-8%）立即記錄為失敗
   - 停損後不再追蹤後續時間軸

3. **更新頻率**：
   - T+1：隔日收盤後更新
   - T+3：第3天收盤後更新
   - T+5：第5天收盤後最終判定

4. **數據來源**：
   - 股價：Yahoo Finance收盤價
   - 法人數據：證交所API + check_institutional.py驗證
   - 量能：券商軟體對照
