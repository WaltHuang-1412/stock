# 歷史驗證檢查清單

⚠️ **強制執行** - 每次盤前分析都必須完成此檢查清單

## 🚨 歷史驗證強制流程

### Step 1: 查詢昨日tracking檔案（強制第一步）

```bash
# 查詢昨日推薦記錄
cat data/tracking/tracking_$(date -d yesterday +%Y-%m-%d).json

# 如果檔案不存在，絕對禁止進行盤前分析
```

**檢查項目**：
- [ ] tracking檔案存在
- [ ] 包含昨日推薦股票清單
- [ ] 包含推薦價格、目標價、停損價
- [ ] 包含推薦理由

### Step 2: 分析昨日推薦表現（強制執行）

**必須建立表格**：

| 股票 | 推薦價 | 收盤價 | 漲跌% | 表現 | 原因分析 |
|------|--------|--------|-------|------|----------|
| XXX | X.XX | X.XX | +/-X.X% | ✅/❌ | 為什麼成功/失敗 |

**檢查項目**：
- [ ] 逐檔查詢昨日收盤價
- [ ] 計算實際漲跌幅
- [ ] 判定成功(✅)或失敗(❌)
- [ ] 分析成功/失敗原因

### Step 3: 計算準確率（強制標示）

**公式**：成功股票數 / 推薦總股票數 = X%

**檢查項目**：
- [ ] 明確標示：「準確率：X/Y = Z%」
- [ ] 如果準確率 <50%，標註⚠️需改進
- [ ] 記錄至 PREDICTION_ACCURACY_TRACKING.md

### Step 4: 成功延續性分析（第二優先）

**針對成功股票**：
1. **成功原因驗證** - 為什麼成功？邏輯是否正確？
2. **延續性檢查** - 成功因素今日是否持續？
3. **相似股票挖掘** - 有沒有類似成功股票的其他標的？

**檢查項目**：
- [ ] 分析每檔成功股票的成功原因
- [ ] 檢查成功邏輯是否延續
- [ ] 今日法人數據是否支持延續
- [ ] 找出具相似特質的新標的

### Step 5: 失敗教訓分析（第三優先）

**針對失敗股票**：
1. **失敗原因分析** - 為什麼失敗？哪裡判斷錯誤？
2. **模式識別** - 是什麼類型的失敗？
3. **避開策略** - 今日如何避開類似錯誤？

**常見失敗模式**：
- ❌ **AI題材鈍化** - AI概念股缺乏新催化劑
- ❌ **估值過高** - 接近高點仍推薦
- ❌ **復甦預期過樂觀** - 產業復甦速度過慢
- ❌ **法人買超力道不足** - 買超金額太小無法支撐

**檢查項目**：
- [ ] 分析每檔失敗股票的失敗原因
- [ ] 歸類失敗模式類型
- [ ] 列出今日需避開的股票類型
- [ ] 調整評分標準避免重複錯誤

### Step 6: 調整今日推薦（最後步驟）

**基於歷史驗證調整**：
1. **優先推薦** - 類似昨日成功股票的標的
2. **降級處理** - 類似昨日失敗股票的標的
3. **加減分機制** - 歷史驗證加減分

**調整機制**：
- ✅ **成功延續型股票** +5-10分
- ❌ **失敗重複型股票** -5-15分
- ⚠️ **不確定型股票** 保持原分數

**檢查項目**：
- [ ] 重新排序推薦優先級
- [ ] 調整推薦股票評分
- [ ] 明確標註歷史驗證邏輯
- [ ] 避開昨日失敗模式股票

## 🔍 歷史驗證輸出格式

### 必須包含的章節

```markdown
## 一、歷史驗證分析（第一優先，30%） 🚨

### 📊 昨日推薦股票表現回顧
[表格]

### 🎯 成功延續性分析
[成功股票分析]

### ❌ 失敗教訓分析
[失敗股票分析]

### 📈 今日策略調整（基於歷史驗證）
[調整邏輯說明]
```

### 推薦股票必須包含歷史驗證邏輯

```markdown
🔥 **歷史驗證邏輯**：類似昨日成功的XXX模式
- ✅ **相似特徵1**（對比說明）
- ✅ **相似特徵2**（對比說明）
- ✅ **相似特徵3**（對比說明）
```

或

```markdown
⚠️ **歷史驗證警示**：類似昨日失敗的XXX模式
- ❌ **風險因子1**（對比說明）
- ❌ **風險因子2**（對比說明）
- ❌ **風險因子3**（對比說明）
```

## ❌ 絕對禁止

1. **❌ 跳過歷史驗證** - 直接從法人數據開始分析
2. **❌ 簡略帶過** - 只用一句話提及昨日表現
3. **❌ 不計算準確率** - 不明確標示成功/失敗比例
4. **❌ 不調整推薦** - 歷史驗證後不改變推薦邏輯
5. **❌ 重複失敗模式** - 明知昨日某類股失敗今日仍推薦
