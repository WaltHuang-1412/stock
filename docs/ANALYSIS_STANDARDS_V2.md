## 🎯 核心理念

```
提前佈局 > 事後諸葛
法人連續買 > 單日買超
投信態度 > 法人買超
還沒漲的 > 已經漲的
客觀數據 > 主觀判斷
全市場掃描 > 只看熟悉股票
```

## 一、盤前分析標準（09:00前完成）

### ✅ 必須執行的步驟

#### Step 1: 查詢連續10天法人數據（強制）

**目的**: 找出法人「持續佈局」而非「單日買超」的股票

**執行方式**:
```python
# 查詢最近10個交易日法人買賣超
dates = 過去10個交易日（排除週末）
for each date:
    查詢證交所API全市場法人數據
    記錄每檔股票的買賣超
```

**篩選條件**:
```
✅ 連續買超天數 ≥ 7天（無賣出）
✅ 累計買超 > 30,000K張
✅ 投信累計買超 > 20,000K張（或佔比>15%）
✅ 力道持續性檢查（重要！）:
   - 計算前6日總買超 vs 近3日總買超
   - 減弱幅度 = (前6日 - 近3日) / 前6日
   - 減弱 < 70% = ✅ 通過
   - 減弱 ≥ 70% = ❌ 力道衰退，不推薦
✅ 投信態度一致（重要！）:
   - 投信累計買超 > 0 = ✅ 投信支持
   - 投信累計賣超 < -100K = ❌ 法人對決，絕對避開
```

**輸出格式**:
```
📊 連續買超股票分析（10/23-11/05）

代號   名稱    買超天數  累計買超   投信累計   近3日力道  評級
2303  聯電    9天      172,688K  +134,989K  持續一致  ⭐⭐⭐⭐⭐
2890  永豐金  9天      121,556K  +34,467K   持續一致  ⭐⭐⭐⭐⭐
...
```

#### Step 2: 交叉比對股價漲跌（強制）

**目的**: 找出「法人在買、但還沒漲」的機會股

**執行方式**:
```python
for each 連續買超股票:
    查詢兩週股價漲跌（起始日-昨日）
    分類：
      - 還沒大漲（<5%）→ 推薦
      - 已經大漲（≥5%）→ 不推薦
```

**輸出格式**:
```
✅ 機會股（還沒大漲 <5%）:
代號   名稱    累計買超   投信態度   兩週漲幅   11/05表現
2303  聯電    172,688K  +134,989K  +3.74%    -0.63%  ⭐⭐⭐⭐⭐
2890  永豐金  121,556K  +34,467K   +1.53%    +0.19%  ⭐⭐⭐⭐⭐

❌ 追高陷阱（已大漲 ≥5%）:
2344  華邦電  121,214K  +58,135K   +20.95%   +5.07%  ❌
2337  旺宏    421,841K  +79,475K   +3.98%    +5.56%  ❌
```

#### Step 3: 檢查用戶持股歷史（強制）

**目的**: 避免推薦用戶有停損歷史或情緒包袱的股票

**執行方式**:
```python
讀取 my_holdings.yaml
for each 推薦股票:
    if 股票 in 用戶歷史:
        if 有停損記錄:
            標記為「不推薦」
            註明原因
```

**範例**:
```
❌ 力積電 (6770) - 不推薦
理由：
- 數據良好（累計+682K張）
- 但用戶11/04停損-5.51%
- 11/05已漲+3.10%
- 若11/06買回 = 追高接回
- 結論：避開情緒包袱
```

#### Step 4: 產出最終推薦（6-8檔）

**推薦格式**:
```markdown
## 🥇 強力推薦 ⭐⭐⭐⭐⭐（1-3檔）

### 1. 聯電 (2303)

**連續買超動態**:
- 連續9天買超（10/23-11/05）
- 累計買超：172,688K張
- 投信累計：+134,989K張（最積極）
- 近3日力道：持續一致

**股價表現**:
- 兩週漲幅：+3.74%（未過熱）
- 11/05表現：-0.63%（法人買但跌 = 提前佈局機會）

**推薦理由**:
✅ 投信最積極（15檔最高）
✅ 連續9天買超，無賣出
✅ 兩週僅漲+3.74%，未過熱
✅ 11/05法人買但跌 = 機會

**進場策略**:
- 進場價位：46.5-47.5元
- 目標價：49-50元 (+5%)
- 停損：46元 (-2%)
- 催化劑：AI晶圓代工需求

**歷史驗證**:
- 過去4次推薦全部失敗（準確率0%）
- 但都是單日買超7-13K張
- 本次累計+172K張，數據更完整
```

### ❌ 絕對禁止

1. **只看單日法人數據**
   ```
   ❌ 錯誤：只看11/05買超TOP10
   ✅ 正確：查詢10/23-11/05連續10天
   ```

2. **推薦已經大漲的股票**
   ```
   ❌ 錯誤：華邦電+5.07%、旺宏+5.56%（11/05當日）
   ✅ 正確：找兩週漲幅<5%的機會股
   ```

3. **推薦有停損歷史的股票**
   ```
   ❌ 錯誤：力積電（用戶11/04停損-5.51%）
   ✅ 正確：優先推薦無不良歷史的標的
   ```

4. **主觀選股**
   ```

   ✅ 正確：全市場掃描977檔連續買超股票
   ```

5. **沒給明確進場價位**

## 二、盤中分析標準（11:00-12:00）

### ✅ 必須執行的步驟

#### Step 1: 只分析盤前預測的6-8檔（強制）

**目的**: 避免事後諸葛

**執行方式**:
```python
預測股票 = 讀取盤前分析的6-8檔
for each 預測股票:
    查詢盤中價格、成交量
    分析量能異常
    對照昨日法人數據
```

**絕對禁止**:
```
❌ 分析盤前沒預測的股票
❌ 說「XX股票+9.84%，因為AI需求」（事後諸葛）
❌ 解釋為什麼沒預測到的股票在漲
```

#### Step 2: 量能異常分析

**目的**: 透過量能推測法人意圖

**量能定義**:
```
爆量：成交量 > 5日均量 3倍
大量：成交量 > 5日均量 2倍
縮量：成交量 < 5日均量 0.5倍
```

**量價關係矩陣**:
| 量能 | 價格 | 昨日法人 | 推測 | 策略 |
|------|------|----------|------|------|
| 爆量 | 大漲 | 買超 | ✅ 法人續買，主升段 | 尾盤可追 |
| 爆量 | 下跌 | 買超 | ⚠️ 法人接刀失敗 | 觀望 |
| 縮量 | 上漲 | 買超 | ⚠️ 量價背離 | 不追 |

#### Step 3: 尾盤策略輸出

**格式**:
```markdown
## 盤中分析 (12:00)

### 預測驗證（只看預測的6-8檔）

#### 聯電 (2303)
- 盤前預測：看漲⭐⭐⭐⭐⭐
- 盤中表現：47.20元 (+0.21%)
- 量能：48,500K（正常）
- 法人意圖：持續佈局中
- 尾盤策略：回檔至47可加碼

### 尾盤佈局建議
✅ 可買：聯電（法人續買）、永豐金（穩定）
⚠️ 觀察：國巨（等盤後法人數據）
❌ 避開：無
```

### ❌ 絕對禁止

1. **事後諸葛**
   ```
   ❌ 錯誤：「創意+9.84%，因為ASIC設計受惠AI」
   ✅ 正確：不提沒預測的股票
   ```

2. **大量解釋失敗原因**
   ```
   ❌ 錯誤：「華邦電預測失敗原因：記憶體產業復甦緩慢...」
   ✅ 正確：「華邦電預測失敗，策略改為觀望」
   ```

3. **追高建議**
   ```
   ❌ 錯誤：「創意可追價1,650~1,680元」（盤前沒預測）
   ✅ 正確：只給預測股的尾盤策略
   ```

## 八、特殊情況處理

### 情況1：用戶持有推薦股票

```
若推薦股票與用戶持股重複：

```

```
⚠️ 暫停推薦
⚠️ 深度檢討分析方法
⚠️ 調整篩選條件
⚠️ 重新驗證後再恢復
```

### 情況3：黑天鵝事件

```
若發生重大突發事件：
⚠️ 在分析中明確標註
⚠️ 降低信心度
⚠️ 提高停損標準
⚠️ 建議保留更高現金比例
```

### 情況4：法人數據異常

```
若發現API數據錯誤：
⚠️ 手動計算：投信+自營商+外資
⚠️ 比對歷史數據合理性
⚠️ 標註「數據存疑」

```

## 十、總結

### ✅ 核心原則（必背）

