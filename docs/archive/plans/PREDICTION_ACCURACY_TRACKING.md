## 💡 關鍵發現

### 1. 法人買超有56.1%預測力

```
這是一個「超過隨機」的數字：
- 隨機猜測: 50%
- 法人買超: 56.1%
- 提升幅度: +6.1個百分點
```

**結論**: 法人買超數據**有效但不夠強**

### 2. 準確率波動很大

```
最低: 40.0% (2025/09/25)
最高: 77.8% (2025/10/28)
波動: 37.8個百分點
```

**結論**: 法人買超的預測力**不穩定**，受市場環境影響

### 3. 近期準確率上升

```
前3天平均 (9/25-10/14): 44.8%
後4天平均 (10/21-10/28): 65.1%
改善幅度: +20.3個百分點
```

**可能原因**:
- 10月市場進入產業輪動期（記憶體、面板）
- 法人佈局方向明確
- AI供應鏈、DRAM漲價等產業趨勢清晰

### 4. 產業趨勢>法人數據

**10/23-10/28高準確率期間的共同點**:
- ✅ 記憶體產業漲價（DRAM Q4+30%）
- ✅ 面板產業價格止跌
- ✅ AI供應鏈訂單明確

**09/25-10/14低準確率期間的特徵**:
- ❌ 無明確產業催化劑
- ❌ 法人試探性買超
- ❌ 金融股輪動失敗

**結論**: **產業趨勢明確時，法人數據預測力提升至70%+**

## 🎯 實戰建議

### ✅ 什麼時候跟著法人買超走

1. **產業趨勢明確時**（準確率70%+）
   - 例：DRAM漲價、AI訂單、面板反彈
   - 法人買超 + 產業邏輯 = 高勝率

2. **連續買超時**
   - 力積電連3日買超 → 10/28成功
   - 友達連2日買超 → 10/29成功

3. **買超>10K張時**
   - 力道夠強，不是試探性買超

### ❌ 什麼時候不跟法人買超

1. **無產業邏輯時**（準確率40-50%）
   - 金融股買超但產業弱勢
   - 個股買超但無催化劑

2. **單日試探性買超**
   - 買超<5K張
   - 昨買今賣的誘多陷阱（康舒案例）

3. **市場環境不佳時**
   - 川習會等重大事件前
   - 大盤系統性風險
