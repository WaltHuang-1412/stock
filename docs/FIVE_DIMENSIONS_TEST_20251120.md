## 🎯 關鍵發現

### 1. 五維度評分有效識別風險

**華邦電案例**：
- 舊系統：法人對決 → 不推薦（正確、但原因錯誤）
- 新系統：46分（法人45、產業40、技術30） → 避開（正確、原因正確）
- **關鍵差異**：新系統識別出「法人力道太小+產業邏輯弱+爆量風險」

### 2. 法人對決不是絕對風險

**外資主導型（6檔、全部上漲）**：
- 南亞、日月光、光寶科、台塑化、聯電、微星
- 共同特徵：外資買>10K、投信賣<10K、時事+產業邏輯強
- 平均評分：78分
- 平均漲幅：+3.1%

**投信主導型（6檔、全部上漲）**：
- 元大金、永豐金、富邦金、彰銀、台新金、凱基金
- 共同特徵：投信買>5K、外資0張或大賣、市場轉強
- 平均評分：64分
- 平均漲幅：+1.7%

**結論**：
- 法人對決≠風險、關鍵看「誰主導+市場環境」
- 外資主導型（平均78分）>投信主導型（平均64分）

### 3. 時事現況權重關鍵

**強勢市場（今天）**：
- 輝達利多+NASDAQ反彈 → 時事現況權重30%
- 結果：時事高分股票（日月光95、南亞86）全部大漲
- 若用弱勢市場權重（時事20%、法人40%）→ 會錯過機會

**驗證**：
```
南亞（強勢市場權重）：
= 70×30% + 86.5×30% + 91×20% + 90×10% + 80×10% = 82.1分（推薦）

南亞（弱勢市場權重）：
= 70×40% + 86.5×20% + 91×10% + 90×10% + 80×20% = 73.1分（僅可考慮）

實際漲幅：+5.76%（強勢市場權重正確）
```

### 4. 單一維度極低分預警有效

**華邦電觸發三重警示**：
1. 法人數據45分（<50分警示）
2. 產業邏輯40分（<50分警示）
