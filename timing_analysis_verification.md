## 🔬 深度發現

### 1. 動能減弱的本質
```
動能減弱（<-30%）= 法人佈局完成 = 準備拉抬

具體特徵：
✓ 爆買後減速（+20K → +10K → +5K）
✓ 最後一天賣超（洗盤）
✓ 連續買超後反轉賣超
✓ 股價還沒漲（低基期）

範例：
- 台積電：法人-27K但ADR+3.42% → 散戶接力
- 旺宏：爆買+59K後-25K → 籌碼穩定 → +9.64%
```

### 2. 動能爆發的陷阱
```
動能爆發（>+100%）= 法人追高 = 隔天沒人接

具體特徵：
✗ 連續加速買超（+10K → +20K → +40K）
✗ 單日爆買>50K
✗ 波動極大（-38K → +59K → -25K）
✗ 股價可能已漲一段

範例：
- 國巨：動能+251.8% → -2.73%
- 力成：動能+187.6% → -0.19%
```

### 3. 臨界值的重要性
```
動能-30%是關鍵臨界值：

成功案例（<-30%）：
- 台積電：-59.0% → +1.11%
- 旺宏：-43.3% → +9.64%

失敗案例（>-30%）：
- 南亞：-22.7% → -4.81%（未達門檻）
- 華航：+25.1% → -2.81%（正值）

結論：必須嚴格執行-30%門檻
```

### 原推薦策略（錯誤）
```python
if 連續買超 >= 3天 and 動能 > +50%:
    推薦()  # ❌ 錯誤：推薦到追高股
```

### 新推薦策略（正確）
```python
# Level 3：佈局完成（強烈推薦）
if 動能 < -30% and (爆買後減速 or 最後一天賣超):
    推薦(倉位='15-20%', 評級='⭐⭐⭐⭐⭐')

# Level 2：佈局中（小倉位）
elif -30% <= 動能 <= +50% and 連續買超:
    推薦(倉位='5-10%', 評級='⭐⭐⭐')

# 避開：追高股
elif 動能 > +100% or 連續加速買超:
    避開()  # ❌ 追高，不推薦
```

## 💡 關鍵洞察

### 顛覆傳統認知
```
傳統認知：法人連續買超 + 動能加速 = 股價會漲
實際結果：法人連續買超 + 動能加速 = 追高，隔天跌

正確認知：法人爆買後減速 + 最後賣超 = 佈局完成，隔天漲
```

### 時機的藝術
```
不是「法人買就推薦」
而是「法人買完了才推薦」

佈局階段：觀察（不推薦）
佈局完成：推薦（動能減弱）
開始拉抬：獲利（動能轉正）
```
